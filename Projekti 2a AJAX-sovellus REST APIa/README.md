# Elokuvatietosovellus

Verkkosovellus, joka näyttää meneillään olevat elokuvat ja niiden tiedot **OMDB API:n** ja **Finnkino API:n** avulla. Voit valita teatterin tarkastellaksesi meneillään olevia esitysaikoja ja elokuvatietoja tai hakea mitä tahansa elokuvaa mukautetulla hakusanalla.

---

## Ominaisuudet

- Valitse teatteri avattavasta luettelosta (Helsinki, Espoo, Tampere).

- Näytä meneillään olevat elokuvat seuraavien perusteella:

- Juliste
- Genre
- IMDB-ikäraja
- Esitysajat
- Mukautettu hakukenttä minkä tahansa elokuvan etsimiseksi nimen perusteella.

- Responsiivinen ruudukkoasettelu elokuvakorteille.

- Selkeä ja käyttäjäystävällinen käyttöliittymä.

---

## Käytetyt teknologiat

- HTML5
- CSS3
- JavaScript
- **OMDB API** elokuvatietojen katseluun
- **Finnkino XML API** elokuvateattereiden katseluun

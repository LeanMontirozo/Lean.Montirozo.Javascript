# TODO-sovellus (jquery)

Tämä on TODO-sovellus, jonka toteutettu HTML,CSS JA JQUERY.
Sovellus tallentaa tehtävät selaimen LocalStorage, joten kaikki tehtävät säilyvät sivun päivittämisen jälkeen.

# Ominaisuudet
- Voi lisätä uuusia tehtäviä
- Listalla merkitseminen valmiiksi tai palauttaminen
- Listalla on poistaminen
- Voi syötteen validointi eli vähintään 3 merkkiä
- Se auttomaattinen tallennus Localstorage tilaan

# Käyttöohjeet
1. Lataa projekti tai kopio html tiedosto.
2. Avaa "index.html" selaimeen.
3. Kirjoita jotain listalla ja paina "Paina" painiketta.
4. Hallitse tehtäviä listassa:
    - "Valmis" se merkitse tehtävän tehtyy
    - "Palauta" merkitsee poistaa valmis-tilan
    - "Poista" merkitse poistaa tehtäviä kokonaan
